## SWP Blog Build


Add Environment Variables
```env
NEXT_PUBLIC_SANITY_PROJECT_ID= # your sanity project id
NEXT_PUBLIC_SANITY_DATASET= # your sanity project dataset
NEXT_PUBLIC_SANITY_API_VERSION= # your sanity api version
```
 Development
```
npm run dev
```

Build
```
npm run build
```


Technologies
- Next.js 13
- React
- Tailwind CSS
- TypeScript
- Sanity v3
